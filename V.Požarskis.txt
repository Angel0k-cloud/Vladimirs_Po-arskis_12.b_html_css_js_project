HTML dokumenta pamata struktūra sastāv no šādiem elementiem:
<!DOCTYPE html> – norāda, ka dokuments ir HTML5
<html> – saknes elements
<head> – satur metadatus (charset, title, CSS pieslēgumu)
<body> – satur lapas redzamo saturu

CSS pievienošana:
CSS ieteicams pievienot kā atsevišķu failu, izmantojot <link> tagu head sadaļā:
<link rel="stylesheet" href="style.css">

JavaScript pievienošana:
JavaScript ieteicams pievienot kā atsevišķu failu ar <script> tagu.
To parasti ievieto pirms </body> taga:
<script src="script.js"></script>
